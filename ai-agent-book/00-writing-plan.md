# 《从零搭建 AI Agent：从 Claude Code 源码到工程实战》写作计划

## 目标

本书面向完全没有 Agent 工程经验的新手，但默认读者至少愿意学习一点 TypeScript、命令行和 API 调用。目标不是只解释某个源码仓库，而是借助 `yasasbanukaofficial/claude-code` 这套 Claude Code 源码快照，系统讲清楚一个工程级 AI Agent 是怎样从零搭建出来的。

最终书稿目标：不少于 50 万字。

由于 50 万字无法在一次模型回复中高质量完成，本书采用连续书稿工程方式推进。每次扩写都遵循同一套目录、术语、代码风格和练习结构，最终合并为完整书稿。

## 写作原则

1. 每章先讲概念，再讲源码对应位置，再讲从零实现，再给练习。
2. 不要求读者一开始理解复杂源码，而是用小项目逐步升级。
3. 所有术语首次出现都要解释。
4. 不直接大段复制 Claude Code 源码，而是讲架构、模式、接口和可复刻实现。
5. 每章必须回答三个问题：为什么需要这个模块，源码里怎么做，自己实现时怎么做。
6. 对安全、权限、上下文、错误恢复等容易忽略的部分要重点讲。
7. 代码示例以 TypeScript 为主，尽量保持最小可运行。
